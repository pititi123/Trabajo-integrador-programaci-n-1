# Gestión de Datos de Países en Python

## Descripción
Aplicación desarrollada en Python para gestionar información de países mediante listas, diccionarios, funciones y archivos CSV.

## Funcionalidades
- Agregar países
- Actualizar datos
- Buscar países
- Filtrar información
- Ordenar registros
- Mostrar estadísticas

## Requisitos
- Python 3.x

## Ejecución
```bash
python main.py
```
